# Maharashtra Election Analysis Dashboard

## Overview
Interactive Excel dashboard analyzing Maharashtra Lok Sabha election results across 48 constituencies.

## Key Features
- **Data Cleaning**: Calculated margins, win types, and asset categories using IF and IFS formulas.
- **XLOOKUP Integration**: Merged election data with regional mapping.
- **Pivot Tables**: Analyzed party performance, regional strength, and win type distribution.
- **Interactive Slicers**: Filter data dynamically by Party and Region.
- **Visualizations**: Column charts, pie charts, and stacked columns.
- **Conditional Formatting**: Heat maps, data bars, and highlights for quick insights.

## Key Insights
- BJP leads with 14 seats, followed by INC with 10 seats.
- 73% of races were Landslides, showing clear voter mandates.
- Western Maharashtra is the largest battleground with 12 seats.
- Only 4 Nail-Biter races occurred across the entire state.

## Tools & Skills Used
- Excel (Formulas: XLOOKUP, IF, IFS)
- Pivot Tables & PivotCharts
- Conditional Formatting
- Interactive Slicers & Dashboard Design
